# Numerical-Integration-of-Atmospheric-Re-Entry-Thermal-Loads
ASTE404 Mini Project: A Python-based numerical simulation of atmospheric re-entry heating using ODE integration to model velocity, altitude, heat flux, and thermal loading along a simplified descent trajectory.

## Requirements
- Python 3.10
- numpy
- matplotlib
- scipy

## Run Instructions 
1) install proper Python packages
       pip install numpy matplotlib scipy
2) locate yourself in the project root directory
3) run the simulation from root folder
       python src/main.py
4) to see changes in timestep - change the vaue of dt in main.py and run again
